# ParamEditor — редактор параметров товара

Проект на React + TypeScript + Vite.  
Позволяет редактировать параметры товара и получать текущую модель через метод `getModel()`.

---

## Скачай репозиторий и установи зависимости

npm install

## Запуск проекта

npm run dev

## Для запуска тестов

npm test

Тесты находятся рядом с компонентами, файлы называются \*.test.tsx.
